** Usage

*** To clone board, run:

#+BEGIN_SRC ruby
ruby -Ilib bin/trello_automation "board_url"
#+END_SRC

*** To clone board for every member on a list

create members list first, format:

John Doe <jdoe>\\
Lorem Ipsum <lipsum>

run:

#+BEGIN_SRC ruby
ruby -Ilib bin/trello_automation "board_url" "path_to_members_list"
#+END_SRC
